 - [x] 半包：公司负责辅材

## 装修流程（毛坯房）

```水 电 木 瓦 油```
这五个步骤遵循谁脏谁先来的原则。

设计量房 -> 主体拆改 -> 水电改造 -> 封埋线槽 -> 防水试验 -> 包管道(静音棉) -> 贴砖 -> 木工 -> 刮腻子 -> 刷墙漆 -> 安装插座面板 -> 厨卫吊顶 -> 安装橱柜 -> 安装成品门 -> 安装灯具 -> 安装洁具 -> 卫浴五金件 -> 安装晾衣架 -> 安装窗轨 -> 安装地板 -> 开荒保洁 -> 家具进场 -> 家电 -> 窗帘/床品 -> 通风

### 硬装阶段
#### 一、主体拆改（7-8天）
装修公司要做的：
* 铲墙皮
* 拆除
* 砌墙
* 封阳台
* 清理垃圾
* 安装中央空调
* 安装地暖

我要做的：
* 联系清运建筑垃圾车
* 马桶、卫浴购买
* 购买中央空调
* 确认地暖铺贴方式

注意事项：
* 做好地漏保护，防止建筑垃圾进入下水道造成阻塞。
* 如果要安装（净水器、饮水机、垃圾处理器等）一起约起来；如果有地暖、新风、和施工方协调先后顺序。
* 需要配备全屋智能设备的，也要在拆除阶段进行前施工。

#### 二、水电改造阶段
装修公司要做的：
* 水电交底
* 电路铺设
* 水路铺设
* 墙排下水改造
* 预埋龙头
* 打机空
* 水电点位核对
* 水电核算

我要做的：

* 过门石、飘窗石材
* 初步了解全屋定制

* 水电完工后，记得找师傅要一份开关、插座、射灯、筒灯数量和尺寸要求统计表，方便后期网购相关材料。

注意事项：

* 是否需要安装过门石，影响到前期找平工作，用极窄压条代替过门石的做法需要地面没有水平高度差
* 厨卫贴砖时，准备好烟道止逆阀。

#### 四、木工阶段（25天左右）
装修公司要做的：

* 眼观预埋
* 灯具开孔位置
* 灯具开孔尺寸
* 测量门
* 木工验收

我要做的：
* 确认前面木饰面、石材、护墙板造型尺寸和位置。

注意事项：
* 主龙骨间距小于1000mm，次龙骨间距小于4000mm，吊点间距小于1000mm，吊顶宽度超过700mm加装主龙骨。

## 瓷砖数量计算

以卫生间长2000mm，宽1600mm，使用300mm * 300mm的瓷砖铺贴地面，使用300mm * 600 mm 铺贴墙面为例。

### 地面瓷砖数量计算

#### 一、使用面积计算
地面面积：S = 长 * 宽（S = 2m * 1.6m=3.2㎡）

计算得到瓷砖数量：`3.2㎡ / 0.09㎡ = 35.55片≈36片`

如果使用面积来计算的话，数量有偏差，你要使用长，宽分别计算。

按10%损耗计算：36片 * 1.1 = 39.6 ≈ 40片

#### 二、使用长宽计算
公式为：`(区域长度/砖长)x(区域宽度/砖宽)`

带入示例，即：(2m / 0.3m) * (1.6m / 0.3m) = 6.67 * 5.33 = 7 * 5.5 = 38.5 ≈ 39片

### 墙面瓷砖数量计算
#### 一、使用面积计算

墙面面积：S = (长 + 宽) * 2 * 高

示例面积：S = (2m + 1.6m) * 2 * 2.5 = 18㎡

计算得到瓷砖数量：18㎡/(0.3m * 0.6m) = 100片

按10%损耗计算：100 * 1.1 = 110片

#### 二、使用长宽高计算

精准计算公式：(区域长/砖长+区域宽/砖长)x2x(区域高/砖宽)

示例计算：(2m / 0.6m + 1.6m / 0.6m) * 2 * (2.5m / 0.3m) = (3.33 + 2.67) * 2 * 8.33 = (3.5 + 3) * 2 * 8.5 = 110.5 ≈ 111片
